# Prompt

This research proposes a neuro-symbolic memory architecture for autonomous agricultural agents that mimics the brain's Complementary Learning Systems to overcome the relational limitations of monolithic memory models like Zep or Mem0. Because standard vector models flatten complex agricultural data and destroy chronological context, this approach decouples memory into distinct, iteratively queried subsystems: a fast-learning "episodic" Knowledge Graph that preserves critical temporal and spatial sequences (like crop cycles or disease spread), and a slow-learning "semantic" machine learning layer that compresses mature data into generalized patterns. By utilizing an agent controller to orchestrate reasoning across these separate processing layers, the framework proves that mimicking human cognitive separation—rather than forcing all data into a single end-to-end model—drastically improves diagnostic accuracy and performance in domain-specific agricultural tasks.
